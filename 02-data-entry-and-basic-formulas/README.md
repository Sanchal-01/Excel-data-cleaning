## 1Ô∏è‚É£ Adjusting Column Size

### Method 1: Manual Resize
- Select the column header (A / B / C)
- Hold and drag the column boundary to adjust width

### Method 2: Auto Resize (Best Practice)
- Double-click the boundary between two column headers
- Column width adjusts automatically based on data length

This helps ensure that **attribute names and values are fully visible**.

---

## 2Ô∏è‚É£ Creating Consecutive Serial Numbers

### Normal Serial Numbers
- Enter first two numbers (e.g., `1`, `2`)
- Select both cells
- Drag the **Fill Handle** to generate the series

### Odd Number Series
- First number: `1`
- Second number: `3`
- Drag the fill handle ‚Üí generates odd numbers

### Even Number Series
- First number: `2`
- Second number: `4`
- Drag the fill handle ‚Üí generates even numbers

### Auto Fill Entire Column
- Enter the first serial number
- Double-click the fill handle
- Excel fills the column based on existing data length

## 3Ô∏è‚É£ Inserting and Deleting Columns

### Insert a Column (Shortcut)
- Select the column where insertion is required
- Press: `Ctrl + +`

### Insert a Column (Normal Way)
- Right-click on column header
- Select **Insert**

---

## 4Ô∏è‚É£ Inserting and Deleting Rows

### Insert a Row
- Select the row
- Press: `Ctrl + +`

### Delete a Row
- Select the row
- Press: `Ctrl + -`

### Select Entire Sheet
- Click on any cell
- Press: `Ctrl + A`

---

## 6Ô∏è‚É£ Changing Date Format
