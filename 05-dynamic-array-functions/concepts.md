## üîπ Dynamic Array Functions (Excel)

In this section, I learned how to use Excel‚Äôs Dynamic Array Functions to perform fast and flexible analysis without writing complex formulas.

Dynamic array functions automatically expand results into multiple cells and update instantly when data changes ‚Äî making reporting truly dynamic.

---

### ‚úÖ Functions Covered

### 1. SORT()

Used to arrange data in ascending or descending order.

**Purpose:**  
Helps quickly organize data (for example: highest to lowest sales).

**Example Use:**
- Sorting marks list  
- Arranging sales data from largest to smallest  

---

### 2. FILTER()

Used to extract data based on conditions.

**Purpose:**  
Returns only those rows that match a given condition.

**Example Use:**
- Showing only pass students  
- Extracting sales of a specific region  

---

### 3. TAKE()

Used to fetch specific rows from the top or bottom of a dataset.

**Purpose:**  
Helps display limited records like top or recent entries.

**Example Use:**
- Top 5 performers  
- Last 3 transactions  

---

### üîÑ Combining Dynamic Functions for Analysis

Dynamic functions become more powerful when used together.

By combining:

- FILTER() ‚Üí to apply conditions  
- SORT() ‚Üí to rank data  
- TAKE() ‚Üí to select top/bottom records  

we can perform **dynamic analysis**.

This allows Excel to automatically update results whenever raw data changes.

---

### üìå Example Use Cases

- Top / Bottom performers  
- Condition-based data extraction  
- Dynamic reporting  

---

### ‚≠ê Key Learning

- SORT arranges data
